# Portfólio – Ensino Médio

Portfólio desenvolvido durante o 3º ano do ensino médio com o objetivo de organizar trabalhos e atividades escolares, integrando front-end, back-end e banco de dados.

## Tecnologias utilizadas
- HTML5
- CSS3
- PHP
- SQL

## Estrutura do projeto
O projeto é composto por várias páginas HTML organizadas por áreas do conhecimento, além de integração com banco de dados para armazenamento de informações.

##  Objetivo
Aplicar conceitos básicos de desenvolvimento web, organização de conteúdo e banco de dados.
